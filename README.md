# Narpay IT Academy

Bu loyiha Narpay IT Akademiyasi uchun yaratilgan veb-sayt hisoblanadi.

## Texnologiyalar

- React
- Vite
- React Router (SPA)

## Ishga tushirish

```bash
npm install
npm run dev
```

## Vercel Deploy Eslatmasi

Vercel uchun `vercel.json` fayli qo'shildi, bu SPA routing (404 xatolik) muammolarini hal qiladi.
